# FUN Consulting

### Professional Business Consulting Services

FUN Consulting adalah perusahaan jasa konsultan profesional yang berfokus membantu perusahaan dalam pengelolaan administrasi bisnis, laporan keuangan, perpajakan, bank garansi, surety bond, legalitas usaha, serta kebutuhan perizinan dan asuransi perusahaan.

Landing page ini dirancang dengan pendekatan modern corporate, enterprise consulting, dan fintech-style premium interface untuk menciptakan kesan terpercaya, profesional, cepat, dan eksklusif.

---

## Local Development

Jalankan live development server:

```bash
npm install
npm run dev
```

Landing page akan tersedia di:

```txt
http://127.0.0.1:5173
```

Untuk akses dari device lain di jaringan yang sama:

```bash
npm run dev:host
```

# Background & Business Needs

Banyak perusahaan mengalami kendala administratif dan finansial yang menghambat fokus utama bisnis mereka.

Mulai dari:

* laporan keuangan yang belum tertata,
* kebutuhan audited report,
* tekanan deadline perpajakan,
* kebutuhan tender mendadak,
* proses bank garansi,
* legalitas usaha,
* hingga proses izin PPIU & PIHK yang kompleks.

Dalam banyak kasus, kondisi tersebut menimbulkan:

* keterlambatan operasional,
* kekhawatiran pemeriksaan,
* kesulitan mengikuti tender,
* hingga hilangnya fokus perusahaan terhadap pertumbuhan bisnis.

FUN Consulting hadir sebagai solusi profesional untuk membantu perusahaan menyelesaikan kebutuhan tersebut secara cepat, aman, terstruktur, dan terpercaya.
